# 🚀 My Machine Learning Project Hub

Welcome! This repository is a collection of all my Machine Learning, Deep Learning, and Computer Vision projects. I built this to showcase my code, track my progress, and save helpful templates.

---

## 🔥 Key Projects inside this Repo

*   **Weapon Identification System:** A computer vision program using YOLO to detect and draw boxes around guns in video streams.
*   **Image Classifier:** A project that uses Transfer Learning to sort pictures into different categories easily.
*   **Data Analysis & Classic ML:** Code files focused on cleaning data, engineering features, and training classic models like Random Forest or XGBoost.

---

## 🛠️ Tools & Technologies Used

*   **Languages:** Python
*   **Libraries:** TensorFlow, PyTorch, Ultralytics (YOLO), Scikit-Learn
*   **Data Cleaning:** Pandas, NumPy, Matplotlib
*   **Deployment:** Streamlit (for building interactive web apps)
