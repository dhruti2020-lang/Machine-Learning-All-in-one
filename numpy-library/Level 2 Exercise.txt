Step 1: Boolean Masking (The "Filter" of Machine Learning)
In machine learning, you often need to isolate specific data points—like finding all houses worth more than $500k, or finding all failing test scores. NumPy lets you filter arrays without writing a single if statement.
The Exercise:
1. Create a 1D array with the values: [12, 45, 7, 89, 23, 56, 3, 90, 34]
2. Create a "mask" (filter) to find all numbers greater than 40.
3. Use that mask to filter your original array so it only prints the numbers greater than 40.


import numpy as np


# 1. Create the array
data = np.array([12, 45, 7, 89, 23, 56, 3, 90, 34])


# 2. Create a boolean mask (This returns an array of True/False values)
mask = data > 40
print("The Mask:", mask) 
# Output looks like: [False  True False  True False  True False  True False]


# 3. Pass the mask into the array using square brackets to filter it
filtered_data = data[mask]
print("Filtered Numbers:", filtered_data) 
# Output: [45 89 56 90]




Step 2: Statistics across Axes (Aggregating Data)
When working with a table of data (2D matrix) in machine learning, rows usually represent samples (e.g., individual patients) and columns represent features (e.g., blood pressure, age, heart rate). You need to know how to find averages or maximums for columns vs. rows.
The Exercise:
Given this 2D matrix representing scores of 3 students across 3 subjects:


scores = np.array([[85, 90, 78],  # Student 1
                   [92, 88, 95],  # Student 2
                   [70, 65, 72]]) # Student 3
1. Calculate the overall average score of the entire class.
1. Calculate the average score for each column (subject) using axis=0.
2. Calculate the maximum score for each row (student) using axis=1.


scores = np.array([[85, 90, 78],
                   [92, 88, 95],
                   [70, 65, 72]])


# 1. Overall average
print("Class Average:", scores.mean())


# 2. Average per column (axis=0 moves vertically down the columns)
print("Average per Subject:", scores.mean(axis=0)) 
# Output: [82.33, 81.0, 81.66]


# 3. Max per row (axis=1 moves horizontally across the rows)
print("Max score per Student:", scores.max(axis=1)) 
# Output: [90, 95, 72]




Step 3: Random Number Generation (Mocking Up Data)
Machine learning algorithms initialize their starting "weights" with random numbers. NumPy has a powerful random module just for this.
The Exercise:
1. Set a random "seed" to 42 using np.random.seed(42). (Seeds ensure that every time you run the code, you get the exact same "random" numbers).
2. Generate a 1D array of 5 random decimal numbers between 0 and 1.
3. Generate a $2 \times 3$ matrix of random integers between 1 and 100.


# 1. Set the seed
np.random.seed(42)


# 2. 5 random decimals between 0 and 1
random_decimals = np.random.rand(5)
print("Random Decimals:\n", random_decimals)


# 3. 2x3 matrix of random integers between 1 and 100
# Note: syntax is np.random.randint(low, high, size=(rows, cols))
random_ints = np.random.randint(1, 101, size=(2, 3))
print("Random Integers:\n", random_ints)




Step 4: Replacing/Capping Values (Data Cleaning)
Sometimes, data has errors or extreme outliers. You might want to say: "If any value is negative, turn it into a 0." NumPy handles this instantly with np.where().
The Exercise:
1. Create this array of balances, some of which are negative errors: balances = np.array([500, -20, 1200, -5, 350])
2. Use np.where() to check the array: if a number is less than 0, replace it with 0. Otherwise, keep the original number.


balances = np.array([500, -20, 1200, -5, 350])


# np.where(condition, value_if_true, value_if_false)
cleaned_balances = np.where(balances < 0, 0, balances)


print("Cleaned Balances:", cleaned_balances)
# Output: [ 500    0 1200    0  350]




Step 5: Matrix Multiplication (The Engine of Neural Networks)
In Level 1, you added arrays position-by-position. But true machine learning models rely on Dot Product matrix multiplication (multiplying rows by columns). This is how Deep Learning neural networks calculate answers.
The Exercise:
Multiply Matrix A by Matrix B using the @ operator or np.dot().


A = np.array([[1, 2],
              [3, 4]])


B = np.array([[5, 6],
              [7, 8]])


A = np.array([[1, 2],
              [3, 4]])


B = np.array([[5, 6],
              [7, 8]])


# Use the '@' symbol for matrix multiplication in Python
result = A @ B


print("Matrix Multiplication Result:\n", result)
# Output:
# [[19 22]
#  [43 50]]
# (How it's calculated: [1*5 + 2*7, 1*6 + 2*8] for the first row)
